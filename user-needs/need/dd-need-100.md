need: dd-need-100
status: proposed
priority: high
name: Publish specified planning data as open data
statement: >
  As a planning data user, I need specified planning application data to be
  published openly and consistently so that I can access, reuse and compare
  planning data without making bespoke requests to individual planning
  authorities.
actors:
  - data-user
  - planning-authority
  - central-government
scope: in
themes:
  - open-data
  - transparency
  - reuse
  - compliance
source:
  - type: legislation
    notes: Levelling-up and Regeneration Act 2023 section 86 provides a power to require specified planning data to be made publicly available under an approved open licence, subject to obligations of confidence and other restrictions on publication.
  - type: data-design
    notes: The June 2026 discussion about the national planning register public view identified a need for a minimum set of planning application data that can be published openly and used to assess initial compliance with the standard.
variations:
next_step:
notes: |
  This need is about publishing a specified subset of planning application data as open data. It does not mean that the complete planning register, submitted application data or validated application data must be published openly.

  Related needs: dd-need-088, dd-need-089.
