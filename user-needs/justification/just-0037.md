The `invalid-from` value lets the planning permission timeline record when the authority treated an application as invalid. Without this value, the specification would not have a standard way to show when an application stopped progressing because it was invalid.

Recording the invalid event with `event-date` and `planning-application` lets users understand when the application journey was interrupted and compare that point with submission, receipt, validation, later information requests or other process events.

This only partially satisfies the need because it records the point at which the authority treated the application as invalid, but does not explain why the application was invalid or what information was missing.
