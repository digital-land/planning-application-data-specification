The `valid-from` value lets the planning permission timeline record the date from which the authority treated an application as valid for statutory or process purposes. Without this value, the decision-stage specification would not have a standard way to show when formal assessment began.

Recording the valid event with `event-date` and `planning-application` lets users compare validation with submission, invalidity, information requests and decision dates so they can understand elapsed time.

This only partially satisfies the need because it records when the application was treated as valid, but does not record any separate validation check date, validation outcome details or accepted application information.
