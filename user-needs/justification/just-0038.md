The `valid-from` value lets the planning permission timeline record the date from which the authority treated an application as valid for statutory or process purposes.
The `validation-completed` value lets the timeline record when the authority completed its validation check.

Without both event values, the decision-stage specification would not distinguish the date the validation check was completed from the date the application was treated as valid from.

Together with `received-date`, these values let users see when formal assessment began and understand how long validation took.
