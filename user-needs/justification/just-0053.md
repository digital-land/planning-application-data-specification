The `planning-performance-agreement` field records whether an application was subject to a Planning Performance Agreement for performance reporting purposes.

The `time-extension-agreed` value in the `permission-process-event` codelist lets the planning permission timeline record when an Extension of Time was agreed.

Together, these fields partially support `need-ps-027` by letting analysts identify applications using Planning Performance Agreements and Extensions of Time. The need is only partially satisfied because this justification does not yet cover applications using the EIA timetable.
