Adding `linked-applications` to the planning-application dataset satisfies the need to link related applications by allowing each application record to reference earlier applications that it directly depends on, varies, amends or seeks to discharge.

This covers direct procedural links such as reserved matters following an outline application, approval of conditions following an earlier permission, section 73 applications and non-material amendments. 

Wider planning history for the same site can still be found through the `site` relationship, rather than listing every application on the same site in this field.
