The `pip` and `technical-details-consent` values in `application-types` identify Permission in Principle applications and Technical Details Consent applications.

Linking a decision notice to the planning application and recording `decision` and `decision-date` lets users track when Permission in Principle and Technical Details Consent applications are decided.

This partially supports tracking progress from Permission in Principle to detailed implementable consent. It is not marked as full because the decision-stage planning application record does not yet expose a relationship from a Technical Details Consent application back to the earlier Permission in Principle application it follows.
