Decision stage datasets

Dataset Description
planning application Records of applications submitted to planning authorities
site Records of development sites linked to planning applications
decision notice The decision notice dataset records the issued decision for a planning application
planning condition Records of individual planning conditions
decision condition Records each planning condition attached to a decision so the decision record reflects all legal requirements
section 106 agreement Records of section 106 agreements linked to developments
planning application document Links a planning application to the documents submitted with it.
planning permission timeline Records key events in the processing of a planning application.
