Track demolition decisions in conservation areas

Satisfied

As a conservation area planner, I need to understand decisions on demolition within conservation areas so that I can manage change that affects the character and appearance of historic places.

Justification

Justification just-0059

The `demolition-con-area` value in `application-types` identifies applications for planning permission for relevant demolition in a conservation area. Linking a decision notice to the planning application and recording `decision` and `decision-date` lets users track decisions on demolition in conservation areas and when those decisions were made.

Satisfaction: full; Confidence: high
