Monitor use of PPAs, EOTs, and EIA timetable

Partially satisfied

As a central government analyst or planning system steward I need to monitor the volume of applications using PPAs, Extensions of Time, or the EIA timetable so that I can understand where additional negotiation, complexity or risk is concentrated in the system and assess whether guidance or support may be required.

Justification

Justification just-0053

The `planning-performance-agreement` field records whether an application was subject to a Planning Performance Agreement for performance reporting purposes. The `time-extension-agreed` value in the `permission-process-event` codelist lets the planning permission timeline record when an Extension of Time was agreed. Together, these fields partially support `need-ps-027` by letting analysts identify applications using Planning Performance Agreements and Extensions of Time. The need is only partially satisfied because this justification does not yet cover applications using the EIA timetable.
