Know when an application was submitted

Satisfied

As a user of the planning system, I need to know when an application was submitted so that I can understand when the clock started and assess timeliness.

Priority
medium
Status
proposed
Themes
processing, transparency
Actors
planning-system-user
Source
ps1-ps2, interview

Justification

Justification just-0036

The `application-submitted` value lets the planning permission timeline record when the applicant submitted the application. Without this value, the decision-stage specification does not have a standard way to represent the submitted date, because submitted date is not otherwise held as a first-class field on another decision-stage dataset. Recording the submitted event with `event-date` and `planning-application` lets users compare submission with later dates, such as receipt, validation and decision, so they can understand when the application journey started and assess elapsed time.
