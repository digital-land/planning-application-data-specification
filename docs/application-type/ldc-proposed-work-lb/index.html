Application schema

Lawful development certficate: Proposed Work to a Listed Building

  • Details
  • Fields
  • Modules
  • Feedback

An application for a certificate confirming whether proposed works to a listed building would be lawful because they do not require listed building consent

Details

Application
ldc-proposed-work-lb
Entry date
2025-08-12
Legislation
  • https://www.legislation.gov.uk/ukpga/1990/9/section/26H
  • https://www.legislation.gov.uk/ukpga/1990/9/section/26I

Fields

These fields are captured at application level before the application-specific modules.

Submission details

Details about the submitted payload, including how it is identified, routed, validated, handled and traced

Reference: submission-details

Required: True

Modules

This application uses the following modules. Each module groups fields that collect a related set of information.

  • Agent contact details (agent-contact)

    Name and contact information if an agent is being used.

  • Agent details (agent-details)

    Name and contact information if an agent is being used.

  • Applicant contact details (applicant-contact)

    Telephone number and email address of the applicant.

  • Applicant details (applicant-details)

    Name and contact information for the parties making the application.

  • Conflict of interest (conflict-of-interest)

    Details of any conflict of interest that may exist between the applicant and planning authority.

  • Checklist (checklist)

    Checking whether all the requirements of the form have been met, such as proof of payment or supporting documentation.

  • Declaration (declaration)

    Signed and dated verification of the application's accuracy.

  • Description of proposed works for listed building lawful development certificate (desc-proposed-works-lb-ldc)

    Details of development plans for the work to the listed building an applicant is seeking a lawful development certificate for

  • Grounds for application (grounds-for-application)

    Why a Certificate of Lawfulness of Propose Works is being requested.

  • Interest details (interest-details)

    Names and contact details for all parties with an interest in the proposed develpoment.

  • Listed building grade (lb-grade)

    The grade of any listed building affected by the proposed development.

  • Pre-application advice (pre-app-advice)

    Details of pre-application advice previously received from the planning authority

  • Site details (site-details)

    Where the proposed development will be built.

  • Site Visit Details (site-visit)

    Information to help the planning authority arrange a site visit
