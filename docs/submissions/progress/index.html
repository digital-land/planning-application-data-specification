Submission coverage

This page shows how much of the planning application volume is covered by the submission specification.

93.6% of planning applications are covered by the submission specification

This means the specification supports application types that account for 93.6% of planning applications.

How this figure is calculated

511,261 of 546,058 applications

are covered by the submission specification.

Based on the 2024 planning application statistics currently available.

What is in scope

Not all planning application types are included in this phase of the submission specification.

You can view the full list of planning application types, including those in and out of scope, in the submission application types list.

Scoping criteria

The submission specification does not currently cover every planning application type. The following criteria were used to decide which application types are in scope for this phase.

  • they had a positive number of applications in the 2024 statistics
  • they are inherited by other application types and needed to support them (for example, prior approval)
  • the 2024 volume is unknown but there is an available application form for the type

Application types with an explicit zero volume are out of scope unless they are inherited by other application types.

Notes

Combined applications

Some planning application forms allow more than one application type to be submitted together.

The submission specification currently defines each application type separately. Applications submitted for a single type, such as householder planning permission or listed building consent, are counted as covered.

Planning application statistics also record cases where multiple application types are submitted together as a combined application. The submission specification does not yet define how these combined submissions should be represented.

For the purposes of the coverage calculation, combined applications are therefore treated as not covered. These volumes will be included once the specification defines how combined applications should be represented.
