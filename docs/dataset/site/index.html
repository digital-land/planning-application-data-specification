Needs satisfied by this dataset

Datasets should only be included in the specification if they satisfy user needs.

dd-need-051 fully satisfied, confidence high

Linked need: dd-need-051

Justification record: just-0027

This dataset is required to satisfy this need.

Recording a site dataset provides a way to show which land or buildings a proposal relates to, helping users understand the location and its context.

The site-boundary field provides the precise spatial extent. The address-text field gives a familiar plain-language locator where the site has a usable address. The description field supports sites that are better identified by a description of the land or buildings.
