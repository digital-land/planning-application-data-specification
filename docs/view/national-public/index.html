National public view

Working draft

The national public view sets out the planning application data that planning authorities are expected to publish as open data.

It includes every listed field for each included record, except that development-scale applies only to full and outline planning applications.

Find out more about the national public view:

  • Find out how the national public view works
  • View the raw schema on GitHub

Datasets to publish

  • Planning application: Publish all records.
  • Site: Publish all records.
  • Decision notice: Publish all records.
  • Planning condition: Publish all records.
  • Decision condition: Publish all records.
  • Section 106 agreement: Publish all records.
  • Planning application document: Include only documents the planning authority has decided may appear on the public register.
  • Planning permission timeline: Include only permission process events selected for the national public view.
