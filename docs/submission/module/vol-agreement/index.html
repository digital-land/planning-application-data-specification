module: vol-agreement

Voluntary agreement

Details of any voluntary agreements made as part of an oil and gas extraction application.

Fields

Draft agreement included
draft-agreement-included

Has an outline or draft agreement been included? (True / False)

Required: True

Datatype: boolean

Expected number: 1

Agreement summary
agreement-summary

Summary of the agreement

Required: False

Datatype: string

Expected number: 1

Rules

  • agreement-summary is required when draft-agreement-included is true
  • Module only applies to extraction-oil-gas application types
