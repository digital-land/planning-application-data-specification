module: proposed-advert-details

Proposed advert details

Details of the proposed advertisements such as their size and how they are made

Fields

Advertisements
advertisements

Structured data about each proposed advertisement

Required: True

Datatype: object

Expected number: 1 or more

Advertisement component

Height from ground
height-from-ground

Height, in metres, from ground to the base of the advertisement

Required: False

Datatype: decimal

Expected number: 1

Height
height

Height, in metres, of dimensions of advertisement

Required: False

Datatype: decimal

Expected number: 1

Width
width

Width of dimensions of advertisement

Required: False

Datatype: decimal

Expected number: 1

Depth
depth

Depth, in metres, of dimensions of advertisement

Required: False

Datatype: decimal

Expected number: 1

Symbol height max
symbol-height-max

Maximum height, in metres, of any individual letters or symbols

Required: False

Datatype: decimal

Expected number: 1

Colour
colour

Colour of proposed sign

Required: False

Datatype: string

Expected number: 1

Materials
materials

Materials of proposed sign

Required: False

Datatype: string

Expected number: 1

Max projection
max-projection

Maximum projection, in metres, of the advertisement from the face of the building

Required: False

Datatype: decimal

Expected number: 1

Illuminated
illuminated

Will the sign(s) be illuminated?

Required: False

Datatype: boolean

Expected number: 1

Illumination method
illumination-method

Method of illumination for the advertisement

Required: False

Datatype: enum

Codelist: illumination-method

Expected number: 1

Illuminance level
illuminance-level

Level of illuminance for the advertisement

Required: False

Datatype: decimal

Expected number: 1

Illumination type
illumination-type

Type of illumination (static or intermittent)

Required: False

Datatype: enum

Codelist: illumination-type

Expected number: 1

Rules

  • At least one advertisement entry must be provided
  • illumination-method is required when illuminated is true
  • illuminance-level is required when illuminated is true
  • illumination-type is required when illuminated is true
  • Dimensional values must be positive numbers
  • illuminance-level must be positive when provided
