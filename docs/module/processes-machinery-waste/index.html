Rules

  • if proposal-waste-management == true then waste-management is required
  • if proposal-waste-management == true then waste-streams is required
  • is-total-capacity-known and is-annual-throughput-known only apply to outline applications

Where this is used

Application types

This module is used in 7 application types:

  • Full planning permission and consent to display advertisements Combined application type
  • Full planning permission and demolition in a conservation area Combined application type
  • Full planning permission
  • Full planning permission and listed building consent Combined application type
  • Outline Planning Permission with All Matters Reserved
  • Outline Planning Permission with Some Matters Reserved
  • Technical details consent
