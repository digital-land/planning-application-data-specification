module: grounds-ldc

Grounds for lawful development certificate

Evidence and explanations relating to a Lawful Development Certificate (LDC) application

Fields

Grounds pre 2024
grounds-pre-2024

List of grounds pre 2024-04-25 under which the certificate is sought

Required: False

Datatype: enum

Codelist: grounds-ldc-pre-apr-2024

Expected number: 1 or more

Grounds post 2024
grounds-post-2024

List of grounds post 2024-04-25 under which the certificate is sought

Required: False

Datatype: enum

Codelist: grounds-ldc-post-apr-2024

Expected number: 1 or more

Other details
other-details

Explanation if other ground is selected

Required: False

Datatype: string

Expected number: 1

Supporting applications
supporting-applications

List of supporting planning permissions, certificates, or notices affecting the application site

Required: False

Datatype: object

Expected number: 1 or more

Supporting applications component

Reference number
reference-number

Reference number of the planning permission

Required: False

Datatype: string

Expected number: 1

Condition number
condition-number

Number of any condition being breached

Required: False

Datatype: string

Expected number: 1

Decision date
decision-date

The date when the decision was made, in YYYY-MM-DD format

Required: False

Datatype: datetime

Expected number: 1

Reason
reason

A textual reason

Required: True

Datatype: string

Expected number: 1

Rules

  • grounds-pre-2024 is not empty OR grounds-post-2024 is not empty
  • if grounds-pre-2024 contains 'other' OR grounds-post-2024 contains 'other' then other-details is required
