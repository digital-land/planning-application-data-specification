module: site-area

Site area

How big the site is including relevant measurements

Fields

Site area in hectares
site-area-in-hectares

The size of the site in hectares

Required: True

Datatype: decimal

Expected number: 1

Site area provided by
site-area-provided-by

Who provided the site area value

Required: False

Datatype: enum

Codelist: provided-by

Expected number: 1

Rules

  • site-area-in-hectares must be a positive number
  • Authority can use site-area-provided-by to determine if calculation verification is needed
  • site-area-in-hectares is measured in hectares
