module: advertisement-types

Advertisement types

What type of advertisements are proposed and how many there will be.

Fields

Advertisement proposal description
description

Description of the advertisement proposal

Required: True

Datatype: string

Expected number: 1

Advertisement proposal type
advertisement-proposal-type

Expected to provide counts for each advertisement type

Required: True

Datatype: object

Expected number: 1 or more

Advertisement proposal type component

Advertisement type
advertisement-type

One of the advertisement-types or other

Required: True

Datatype: enum

Codelist: advertisement-type

Expected number: 1

Advertisement count
advertisement-count

Number of this type of advertisement

Required: True

Datatype: integer

Expected number: 1

Advertisement other description
advertisement-other-description

Details required if other advertisement type is selected

Required: False

Datatype: string

Expected number: 1

Rules

  • At least one advertisement-proposal-type entry must be provided
  • advertisement-other-description is required when advertisement-type is 'other'
  • advertisement-count must be a positive integer

Where this is used

Application types

This module is used in 2 application types:

  • Advertising consent
  • Full planning permission and consent to display advertisements Combined application type
