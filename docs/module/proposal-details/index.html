module: proposal-details

Description of the proposal

What development, works or change of use is proposed

Fields

Proposal description
description

A description of what is being proposed, including the development, works, or change of use

Required: True

Datatype: string

Expected number: 1

Reserved matters
reserved-matters

Identifies which reserved matters are being submitted for approval as part of this application

Required: True

Datatype: enum

Codelist: reserved-matter-type

Expected number: 1 or more

Related application
related-application

Details about the approved development, as shown in the decision letter

Required: True

Datatype: object

Expected number: 1

Related application details component

Reference
reference

The reference for the related application

Required: True

Datatype: string

Expected number: 1

Description
description

A description of the related application

Required: True

Datatype: string

Expected number: 1

Decision date
decision-date

The date when the decision was made, in YYYY-MM-DD format

Required: False

Datatype: datetime

Expected number: 1

Proposal started
proposal-started

Has any work on the proposal already been started

Required: True

Datatype: boolean

Expected number: 1

Proposal start date
proposal-started-date

The date when work on the proposal started, in YYYY-MM-DD format

Required: False

Datatype: datetime

Expected number: 1

Proposal completed
proposal-completed

Has any work on the proposal already been completed

Required: True

Datatype: boolean

Expected number: 1

Proposal completion date
proposal-completed-date

The date when work on the proposal was completed, in YYYY-MM-DD format

Required: False

Datatype: datetime

Expected number: 1

PIP reference
pip-reference

Reference number for the Planning in Principle (PIP) application this relates to

Required: False

Datatype: string

Expected number: 1

Is public service infrastructure
is-psi

For applications made on or after 1 August 2021, is the proposal for public service infrastructure development

Required: True

Datatype: boolean

Expected number: 1

Rules

  • description must be clear and concise
  • proposal-started-date must not be in the future
  • proposal-completed-date must be after proposal-started-date if both provided
  • reserved-matters must be valid types from the codelist
  • related-application reference must exist in authority records
  • pip-reference must match an existing Planning in Principle application
  • PSI projects must be checked against infrastructure improvement plans
