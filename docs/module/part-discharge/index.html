module: part-discharge

Part discharge

Details of how the applicant is meeting a specific part of a set of conditions made by the planning authority.

Fields

Is discharging part
is-discharging-part

Is applicant trying to discharge part of condition?

Required: True

Datatype: boolean

Expected number: 1

Discharging part details
discharging-part-details

Indicate which part of the condition the application relates to

Required: False

Datatype: string

Expected number: 1

Rules

  • discharging-part-details is required when is-discharging-part is true
  • discharging-part-details should specify which part of the condition is being addressed
