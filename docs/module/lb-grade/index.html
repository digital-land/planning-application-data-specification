module: lb-grade

Listed building grade

The grade of any listed building affected by the proposed development.

Fields

Listed building grade
listed-building-grade

The grade of the listed building, selected from the listed-building-grade codelist or "don't know"

Required: True

Datatype: enum

Codelist: listed-building-grade

Expected number: 1

Listed building
listed-building

Listed building reference for cross-referencing with listed building records

Required: False

Datatype: string

Expected number: 1

Provided by
provided-by

Source of the listed building grade information

Required: False

Datatype: enum

Codelist: provided-by

Expected number: 1

Rules

  • listed-building-grade must be selected from the listed-building-grade codelist or 'don't know'
  • If listed-building is provided, it must reference a valid listed building
